Este documento tem como objetivo identificar e analisar os **principais stakeholders** do projeto *Invitation Tracker*, avaliando seus níveis de **influência, interesse e estratégias de engajamento**.

## 3. Matriz de Influência x Interesse

- **Alta influência / Alto interesse** → Gerente de Projeto, Administrador, Patrocinador
- **Alta influência / Baixo interesse** → Instituição de apoio ou gestor externo
- **Baixa influência / Alto interesse** → Equipe de Campo, Convidados
- **Baixa influência / Baixo interesse** → Usuários temporários de teste

**Estratégias por quadrante:**

- **Alta Influência / Alto Interesse** (Gerenciar de Perto): Comunicação frequente, envolvimento nas decisões
- **Alta Influência / Baixo Interesse** (Manter Satisfeito): Informar sobre decisões importantes, evitar sobrecarga
- **Baixa Influência / Alto Interesse** (Manter Informado): Comunicação regular, coleta de feedback
- **Baixa Influência / Baixo Interesse** (Monitorar): Acompanhamento mínimo necessário
